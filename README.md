## Getting started

```bash
git clone https://github.com/rajat-np/jwt-auth-express
cd jwt-auth-express
npm install
cp .example.env .env
npm start
```

Edit the `.env` file to set the correct env variables.

## Usage

To run development server
<br/>

```bash
npm start
```

To run test suite
<br/>

```bash
npm test
```

To run swagger API documentation
<br/>

```bash
npm run swagger
```
